Archive USB Format Free

Simple, efficient Python app to format USB drives (FAT32 or NTFS) with auto-detect and a friendly GUI.

Author: Bocaletto Luca

License: GPLv3  |  View LICENSE

Source: GitHub Repository

Key Features

  • Format USB drives quickly (FAT32 or NTFS)
  • Automatically detects connected USB drives
  • Easy-to-use graphical interface (tkinter)
  • Cross-platform: Windows, Linux, macOS
  • Distributed under GPLv3 for freedom and customization

System Requirements

  • Python 3.7 or newer
  • tkinter (standard library GUI toolkit)
  • psutil (for drive detection)
  • subprocess (standard library)

Installation

# Clone the repository
git clone https://github.com/bocaletto-luca/Archive-USB-Format.git
cd Archive-USB-Format

# Install dependencies
pip install psutil

How to Use

  1. Launch the app: 
    python main.py
  2. Select your file system (FAT32 or NTFS) from the dropdown
  3. Choose the USB drive to format
  4. Click Format to begin
  5. View the result message in the application window
